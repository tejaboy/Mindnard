# Mindnard
Open source visual programming software for ev3dev (Python).

## Getting Started

### Prerequisites
1. Adobe Flash Professional or Adobe Animate

### Installing
1. Clone the repo to your local machine.
2. Open up Mindnard.fla
3. Press CTRL+F5 to run the application
